**Pago en Línea (Stripe Checkout)**: Este componente permite realizar pagos en línea de forma segura mediante Stripe, ya sea para registrar una nueva empresa, renovar un plan o completar el registro gratuito (plan Demo). Flujo de funcionamiento: Selección de plan: Se obtiene desde la URL (plan) y puede ser "Demo", "Negocios" o "Inteligente". Plan Demo: No requiere pago. Solo se registra la empresa y se envía un correo de bienvenida con contraseña por defecto. Pago con tarjeta (Stripe): Se genera un PaymentIntent en el backend con el monto correspondiente. El usuario ingresa el nombre en la tarjeta y los datos del plástico. Stripe procesa el pago y, si es exitoso: Se registra la empresa (o se actualiza si es renovación). Se guarda la transacción en el backend. Se notifica al usuario con un mensaje de éxito. Renovación de empresa: Si la URL contiene renew=1, se actualiza la fecha de vencimiento un mes más desde la fecha actual. Se requiere el parámetro empresaId para identificar la empresa a renovar. Características clave: Integración directa con Stripe (@stripe/react-stripe-js y loadStripe). Flujo unificado para registro y renovación. Uso de localStorage para recuperar los datos capturados previamente en el registro. Guardado de transacción detallada en el backend (incluyendo tarjeta, monto y estado). Envío automático de correo de bienvenida tras el registro. Interfaz clara y responsive con mensajes contextuales según la acción realizada.
