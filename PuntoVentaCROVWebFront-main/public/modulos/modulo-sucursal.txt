**Editar Sucursal**: Permite visualizar y modificar los datos de una sucursal específica a través de un formulario estructurado con campos para razón social, RFC, contacto, dirección, correo, teléfonos, nombre comercial y giro comercial. Al cargarse, consulta la API para obtener la información actual de la sucursal vinculada al usuario. Ofrece una lista predefinida de giros comerciales para su selección. Integra validación básica y envía los cambios mediante una petición PUT, mostrando notificaciones de éxito o error según el resultado. La interfaz es clara, con controles accesibles y una experiencia fluida para la gestión de información empresarial.
