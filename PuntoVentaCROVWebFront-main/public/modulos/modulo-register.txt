**Registro de Empresa**: Este componente guía a nuevos usuarios a través del proceso de registro de una empresa en la plataforma. Está dividido en 5 pasos interactivos y secuenciales. Flujo paso a paso: Paso 1: Verificación de Correo El usuario ingresa un correo electrónico. Se verifica en el backend si ya está registrado. Si es válido, se envía un código de verificación al correo y se avanza al paso 2. Paso 2: Confirmación de Código El usuario debe ingresar el código recibido. Si el código es correcto, se avanza al paso 3. Paso 3: Datos de la Empresa El usuario llena un formulario con datos de la empresa, sucursal y contacto. Los campos obligatorios son: nombre comercial y celular. Si están completos, se pasa al paso 4. Paso 4: Selección de Plan Se muestran 3 paquetes con sus beneficios: Demo: $0 – 30 días gratis, funciones básicas. Negocios: $299 – con soporte 24/7 y reportes. Inteligente: $499 – incluye IA para sugerencias y análisis. El usuario elige un plan y se hace lo siguiente: Si es Demo, se realiza el registro directo (/auth/setup) y se envía el correo de bienvenida. Si es Negocios o Inteligente, se guarda la info en localStorage y se redirige a la página de pago (/payment). Paso 5: Confirmación Final Se muestra un mensaje de éxito. Se informa que se ha enviado un correo con las credenciales. Se ofrece un enlace para iniciar sesión. Características clave: Verificación de correo mediante código. Registro multi-paso con validación progresiva. Selección de plan con diseño scrolleable e interactivo. Uso de localStorage para persistir temporalmente datos entre pasos. Envío automático de correo de bienvenida tras el registro. Redirección al flujo de pago si el plan no es gratuito.
