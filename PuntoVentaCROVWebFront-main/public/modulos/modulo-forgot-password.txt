**Recuperación de Contraseña**: Este componente permite a los usuarios restablecer su contraseña de forma segura mediante un flujo en tres pasos: verificación de correo, ingreso de código y asignación de una nueva contraseña. Flujo de funcionamiento: Correo: El usuario ingresa su correo electrónico. Se valida que exista en el sistema antes de enviar un código de verificación por email. Código: El usuario ingresa el código recibido. Si coincide con el enviado, puede continuar al siguiente paso. Nueva contraseña: El usuario establece y confirma su nueva contraseña. Si las contraseñas coinciden, se actualiza exitosamente en la base de datos. Características clave: Validación de existencia del correo antes de enviar el código. Comparación exacta entre código enviado y recibido. Validación de coincidencia entre contraseña nueva y confirmación. Mensajes de error claros en cada etapa. Flujo paso a paso guiado, con interfaz sencilla y centrada.
